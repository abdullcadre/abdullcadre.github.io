# Portfólio Abdullcadre.github.io

Este é o repositório do meu portfólio hospedado no GitHub Pages. O site foi criado com o software Nicepage e contém informações sobre mim, meus projetos e habilidades.

## Acesso ao site

Você pode acessar o meu portfólio através do link https://abdullcadre.github.io/

## Como foi criado

O site foi criado utilizando o software Nicepage, uma ferramenta de criação de sites visual que permite a criação de sites responsivos sem a necessidade de codificação. O código-fonte do site está hospedado neste repositório do GitHub.

## Personalização

Se você deseja personalizar este site e criar o seu próprio portfólio, siga os passos abaixo:

1. Faça o download do software Nicepage em https://nicepage.com/.

2. Abra o software Nicepage e selecione o modelo de portfólio que deseja personalizar.

3. Personalize o modelo conforme suas necessidades, inserindo suas informações, projetos e habilidades.

4. Exporte o site para uma pasta local em seu computador.

5. Crie um novo repositório no GitHub e faça o upload dos arquivos exportados para este repositório.

6. Ative o GitHub Pages nas configurações do repositório para hospedar seu portfólio.
